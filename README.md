# TP14 - GraphQl

Objectifs
---------
- Présenter le projet et son but.
- Expliquer comment exécuter ou visualiser le projet localement.
- Conserver les captures d'écran montrant les différents états/fonctionnalités.
- Faciliter la contribution et la lecture du projet.

Prérequis
---------
- git (pour cloner le dépôt)
- Un éditeur ou IDE (VS Code, IntelliJ, etc.)
- Java 21

Installation
-------------------------------
1. Cloner le dépôt :
```bash
git clone https://github.com/RadouaneAitSaid123/Tp14-architecture.git
cd Tp14-architecture
```
```bash
mvn clean install
mvn spring-boot:run   # ou la commande adaptée
```
